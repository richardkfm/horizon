Import a how-to page

Fetches a WikiHow-shaped page (an intro plus numbered steps) and saves it as a guide. Step images are downloaded too, so the guide works offline.

WikiHow content is CC BY-NC-SA; check the source's licence before sharing or redistributing an imported guide further.

Import a book

Splits a plain-text or Markdown book into one guide per detected chapter (Markdown headings, or "Chapter N" / "Part N" lines).

A book may be copyrighted — check that you have the right to import and share it before uploading.
