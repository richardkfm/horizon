# Communicate with signs, gestures, and pictograms

Not all communication is spoken. Some neighbours are deaf or hard of hearing;
some situations are too loud, too far, or too urgent for voices; and some
information — where the safe water is, what's dangerous — should be readable
by anyone who walks past, whatever language they speak. Hands and pictures
cover all three.

> **Note.** Sign languages are real, full languages — and each region has its
> own, as different from each other as spoken languages are. The best teacher
> is a deaf person in your own community; learning even basics from them says
> "you belong here" better than anything else you can do.

## Communicate well with deaf and hard-of-hearing neighbours

- **Get attention respectfully:** a wave in their field of view, a gentle tap
  on the shoulder, or a flick of the lights — not shouting or grabbing.
- **Face the person, with light on your face.** Lip-reading and expression
  need a clear, lit view; don't cover your mouth or turn away mid-sentence.
- **Speak normally, don't exaggerate.** Over-mouthed or shouted words are
  *harder* to read. Rephrase rather than repeat the same missed word.
- **Write and draw freely.** A small slate or notepad beats struggling —
  writing is not a lesser way to talk.
- **In groups, take turns and point to who's speaking** so the conversation
  can be followed; brief the person on the topic when they join.
- **Learn from them:** ask which signs their sign language uses for the words
  your group needs most, and let them correct your attempts.

## Agree working signs for the group

For work parties, lookouts, and anywhere voices don't carry, agree a small
set of unambiguous hand signals — then practise them until they're automatic:

- **stop** · **come** · **wait** · **help** · **danger** · **yes / no** ·
  **water** · **food** · **look at me** · **all clear**
- Make each sign big, distinct, and readable at distance and in poor light.
- Test them across the real distance you'll use them (field to fence,
  rooftop to street) before you rely on them.
- Teach every newcomer and child the same set — a signal only half the group
  knows is worse than none.

## Post pictograms anyone can read

A good pictogram tells a stranger what they need to know with no words at
all. Use them on water points, stores, first-aid kits, toilets, and hazards.

```ascii
   safe to drink        NOT for drinking       first aid        toilet
    _________            _________            _________       _________
   |         |          |    _    |          |         |     |         |
   |   /\    |          |   / \   |          |    _    |     |   ( )   |
   |  /  \   |          |  / X \  |          |  _| |_  |     |    |    |
   |  \  /   |          |  \ X /  |          | |_   _| |     |   /|\   |
   |   \/    |          |   \_/   |          |   | |   |     |   / \   |
   |  (cup)  |          | (cross- |          |   |_|   |     |         |
   |_________|          |  ed out)|          |_________|     |_________|
                        |_________|
```
*Four signs worth posting first: safe water, unsafe water, first aid, toilet —
simple shapes, thick strokes, one idea per board.*

- **One idea per sign.** A pictogram that needs explaining has failed.
- **Use colour and shape together** (red + crossed-out for "no"), never colour
  alone — some people don't see colour differences.
- **Keep them consistent everywhere:** same symbol, same meaning, at every
  water point and store. Post a small key board where people gather.
- **Draw big, thick, and high-contrast** so signs read at a glance, at
  distance, and at dusk.

> **Do now.** If your group shares any water source, mark safe and unsafe
> water with unmistakable, agreed signs today — it's the pictogram that
> prevents the most harm (links to testing whether water is safe to drink).

## Keep it growing

Add signs and symbols as needs appear, retire ones that confuse, and record
the agreed set on your group's word list so it survives beyond the people who
invented it (links to communicating across a language barrier and to
preserving knowledge through storytelling).
